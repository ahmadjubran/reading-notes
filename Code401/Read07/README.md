# Chakra UI

## What is a Chakra UI?

Chakra UI is a simple, modular and accessible component library that gives you the building blocks you need to build your React applications.

## Is Chakra UI better than material UI?

- Chakra UI is a simple, modular and accessible component library that gives you the building blocks you need to build your React applications.
- Material UI is a component library that provides you with pre-built components that you can use to build your application.

Chakra UI is better than material UI because it is more simple and modular. It also has more accessibility features. Chakra UI is also more customizable than material UI.

## How to use Chakra UI?

- Install Chakra UI using npm or yarn.
- Import the Chakra UI components you want to use.
- Use the components in your app.
